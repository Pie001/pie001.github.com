# 일단 메모

### 반복문 (map)
#### 예제 1
```js
<ul>
    {['1','2','3','4'].map((v) => {
        return (
        <li>{v}</li>
        );
    })}
</ul>
```
- 결과
```
1
2
3
4
```

#### 예제 2
```js
<ul>
    {[['사과', '빨강'],['바나나', '노랑'],['포도', '보라'],['귤', '오렌지']].map((v) => {
        return (
        <li>{v[0]} - {v[1]}</li>
        );
    })}
</ul>
```
- 결과
```
사과 - 빨강
바나나 - 노랑
포도 - 보라
귤 - 오렌지
```

### 반복문 (key)
리액트가 vue.js에 비해 반복문이 좀 깔끔하지 않다는듯.

#### 예제 1
```
<ul>
    {[{fruit: '사과', color: '빨강'},{fruit: '바나나', color: '노랑'},{fruit: '포도', color: '보라'},{fruit: '귤', color: '오렌지'}].map((v) => {
        return (
        <li>{v.fruit} - {v.color}</li>
        );
    })}
</ul>
```
- 결과
```
사과 - 빨강
바나나 - 노랑
포도 - 보라
귤 - 오렌지
```
