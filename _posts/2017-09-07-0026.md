## Ajax로 리스트 오브젝트를 Controller로 전달하기

### JS
```javascript
var things = [
    { id: 1, color: 'yellow' },
    { id: 2, color: 'blue' },
    { id: 3, color: 'red' }
];
$.post('@Url.Action("PassThings")', { things: things },
   function () {
        $('#result').html('"PassThings()" successfully called.');
   });
```

### Controller
```cs
[HttpPost]
public void PassThings(IEnumerable<Thing> things)
{
    // do stuff with things here...
}
```
